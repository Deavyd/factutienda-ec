# Base de Datos - FactuTienda EC

## Tablas principales
- empresas
- establecimientos (sucursales)
- puntos_emision
- personas (clientes + proveedores)
- productos
- facturas + detalle_facturas
- kardex (movimientos de inventario)
- compras
- notas_credito
- usuarios

## Reglas importantes
- Un solo certificado digital para toda la empresa
- Secuencial independiente por punto de emision
- Stock centralizado inicialmente
